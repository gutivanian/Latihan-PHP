Konstanta

Variabel di PHP sifatnya mutabel, atau dapat diubah. Jika kita ingin membuat variabel yang imutabel/tidak bisa diubah, maka kita tidak bisa dilakukan di PHP. Sebagai gantinya, terdapat fitur yang mananya "constant". constant adalah tempat untuk menyimpan data yang tidak bisa diubah setelah dideklarasikan. Untuk membuat constant, kita bisa menggunakan fungsi "define()". Biasanya nama konstanta ditulis dalam UPPER_CASE. 

        <?php
            define('AUTHOR', 'programmer zaman now');
            define('APP_VERSION', 100);
            // define('APP_VERSION', 500); // akan error tidak dapat deklarasi ulang

            echo AUTHOR;
            echo "\n";
            echo APP_VERSION;
            echo "\n";
