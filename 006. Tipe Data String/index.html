String

Tipe data string adalah tipe data representasi dari teks. String bisa mengandung string kosong atau teks. Penulisan string bisa menggunakan kutip satu atau kutip dua. 

        <?php
            echo "ini string";
            echo "Ini juga string 2";
            echo '';

Kelebihan menggunakan kutip dua, kita bisa menggunakan "escape sequence", seperti "\n" untuk enter atau "\t" untuk tab. 

        <?php
            echo "ini \n enter";
            echo "ini \t tab";
            echo "ini \" dikutip \"";

Multiline String

Kadang kita ingin membuat data string yang lebih dari satu baris. Untuk melakukannya, kita bisa menggunakan "\n". Namun php memiliki fitur berguna bernama "Heredoc" dan "Nowdoc".

Heredoc

Heredoc adalah fitur untuk membuat teks panjang, sehingga kita tidak perlu manual melakukan enter tab dan lainnya. Cara penulisan heredoc adalah dengan "<<< NamaTeks" kemudian enter dan diisi teksnya lalu diakhiri dengan namateks kembali. 

        <?php
            echo <<<TEKS
            ini adalah teks yang sangat panjang. 
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Veritatis, velit fugit? Impedit optio consectetur ipsa possimus accusantium perspiciatis corrupti cum, assumenda, perferendis deleniti placeat quae sint quam quisquam error suscipit.
            TEKS;

Nowdoc

Nowdoc adalah fitur yang sama dengan Heredoc, hanya saja nowdoc tidak dapat menggunakan "escape sequence". Untuk nowdoc, nama teks diapit petik satu. 

        <?php
            echo <<<'TEKS'
            ini adalah teks yang sangat panjang. Lorem ipsum dolor sit amet, consectetur adipisicing elit. Veritatis, velit fugit? Impedit optio consectetur ipsa possimus accusantium perspiciatis corrupti cum, assumenda, perferendis deleniti placeat quae sint quam quisquam error suscipit.
            TEKS;
