File PHP

File kode PHP diakhiri dengan ekstensi .php. Di awal kode program PHP wajib menambahkan <?php dan diakhir kode perlu ditambahkan ?>, namun tidak wajib. Nama file PHP tidak ada aturan, namun untuk mempermudah jangan menggunakan spasi. Untuk menampilkan tulisan di PHP, kita bisa menggunakan perintah "echo". 

        <?php

        echo "Hello World"

yang disimpan di file "hellow-world.php", maka kita dapat hmenjalankan perintah di cmd "php hellow-world.php
