PHP

PHP adalah kode yang dieksekusi pada server. PHP berguna untuk wordpress. PHP cukup dalam untuk menjalankan jaringan sosial yang besar. PHP juga mudah dipelajari bagi yang baru mempelajari bahasa untuk server.

File PHP

  1. File PHP dapat berisi teks, HTML, CSS, Javascript dan sintaks PHP
  2. Kode PHP dieksekusi di server dan hasilnya dikembalikan ke browser sebagai HTML biasa.
  3. File PHP memiliki ekstensi ".php"

Kemampuan PHP

  1. PHP dapat membangkitkan konten halaman dinamis
  2. PHP dapat membuat, membuka, membaca, menulis, menghapus, dan menutup file pada server
  3. PHP dapat mengkoleksi data form
  4. PHP dapat mengirim dan menerima cookies
  5. PHP dapat menambahkan, menghapus, dan memodifikasi data pada database
  6. PHP dapat digunakan untuk mengontrol akses pengguna
  7. PHP dapat mengenkripsi data

Dengan PHP kita tidak dibatasi untuk menghasilkan output HTML, kita juga dapat menghasilkan output gambar dan file PDF. Kita juga dapat membuat output teks lain seperti XHTML dan XML.

Alasan Menggunakan PHP

  1. PHP dapat berjalan di bermacam-macam platform (Windows, Linux, Mac, dll)
  2. PHP kompatibel terhadap semua server yang ada sekarang (Apache, IIS, dll)
  3. PHP dapat mendukung banyak tipe database
  4. PHP gratis
  5. PHP mudah dipelajari dan berjalan dengan efisien di server

PHP 7

  1. PHP 7 lebih cepat dari PHP sebelumnya
  2. Memperbaiki penanganan error
  3. Mendukung deklarasi tipe dengan lebih ketat untuk argumen fungsi
  4. Mensupport operator baru seperti operator spaceship "<=>
