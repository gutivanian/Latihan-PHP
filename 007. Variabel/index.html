Variabel

Variabel adalah tempat untuk menyimpan data sehingga bisa digunakan lagi di program selanjutnya. Di PHP, variabel bisa menampung berbagai jenis tipe data dan bisa berubah ubah tipedatanya. Untuk membuat variabel, kita bisa menggunakan tanda "$" dollar diikuti dengan nama variabelnya. Penamaan variabel tidak boleh mengandung spasi. 

        <?php
            $name = "Eko";
            $age = 30;

            echo "Name: ";
            echo $name;
            echo "\n";
            echo "Age: ";
            echo $age;

Variabel dari Variabel

PHP memiliki kemampuan "variable variables", yaitu membuat variabel dari nilai string variabel. Walaupun fitur ini ada, ini jarang digunakan karena cukup membingungkan. Untuk menggunakannya, gunakan dollar ganda $$ diikuti dengan nama variabel. 

        <?php
            $nama = "Saya";
            $$nama = "Anda";

            echo "\$name = ";
            echo $nama;
            echo "\n";
            echo "\$Saya = ";
            echo $Saya;
