Cara Menggunakan PHP

Untuk menggunakan PHP, ada beberapa yang dibutuhkan:

  1. Mencari Web Host yang mendukung PHP dan MySQL
  2. Menginstall sebuah web server pada komputer sendiri dan kemudian install PHP dan MySQL

Menggunakan Web Host Yang Mendukung PHP

Jika server kita telah mengaktifkan dukungan ke PHP, kita tidak perlu melakukan apapun. Kita hanya perlu membuat file dengan ekstensi ".php", kemudian letakkan pada direktori web dan server akan secara otomatis melakukan parsing. Kita tidak perlu melakukan compile atau install alat lain. Karena PHP gratis, kebanyakan web host menawarkan dukungan PHP.

Instalasi XAMPP

Cara install PHP di Windows

  1. Install XAMPP
  2. Pergi ke direktori php di folder XAMPP
  3. Tambahkan alamat direktori php ke environment
  4. php siap digunakan
