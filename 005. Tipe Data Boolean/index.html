Tipe Data Boolean

Tipe data boolean adalah tipe data paling sederhana di PHP. Tipe data ini hanya memiliki dua nilai, yaitu true dan false. Tipe data boolean case insensitive (tidak bergantung sama huruf besar/kecil) 

        <?php
            echo "Benar: ";
            var_dump(true);
            echo "Salah: ";
            var_dump(false);
