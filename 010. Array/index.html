Array

Array adalah tipe data yang berisikan nilai kosong atau beberapa data. Array di PHP bisa berisikan data dengan tipe data yang berbeda-beda. Array di PHP memiliki panjang dinamis, artinya kita bisa menambah data ke array sebanyak-banyaknya tidak dibatasi kapasitasnya. Indeks array diawali dengan nilai 0. Ada dua cara membuat array, yaitu dengan fungsi array(nilai) atau dengan kurung kotak [nilai]. 

        <?php
            $nilai = array(1,2,3,4);
            var_dump($nilai);

            $nama = ["Nama 1", "Nama 2", "Nama 3"];
            var_dump($nama);
